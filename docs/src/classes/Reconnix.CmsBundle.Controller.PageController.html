\Reconnix\CmsBundle\Controller

\Reconnix\CmsBundle\ControllerPageController

Renders a Page on the website.

Defines all common functionality required amongst the CMS pages. Technically this acts more like a base 'model' and not a 'controller'. It is not called directly from any routing configuration.
