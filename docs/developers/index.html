Reconnix CMS Basics

The application has a "front" end and a "back" end. The back end is accessed from /admin. The purpose of the admin area is to provide CRUD functionality to the content that is displayed on the front end. The front end web pages are accessed from /page/{page-name}, where /page refers to the Page Controller, and {page-name} is a string variable passed to the IndexAction function of the Controller:
